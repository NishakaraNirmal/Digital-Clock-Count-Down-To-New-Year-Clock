# Digital Clock And Count Down To New Year Clock.


## Live Demo
**Click here to view the running clock and countdown!**
[VIEW LIVE PROJECT HERE] (https://nishakaranirmal.github.io/Digital-Clock-Count-Down-To-New-Year-Clock/).

**This project is a simple, dark-themed digital clock that displays the current time in real-time and includes a countdown to the New Year. It was built using standard HTML, CSS, and JavaScript.


## Features
* Real-time digital clock display.
* Accurate countdown timer to January 1st.
* Responsive design.

## Technologies Used

* **HTML(`index.html`):** For the structure and content.
* **CSS (`style.css`):** For styling, layout, and the dark theme.
* **JavaScript (`script.js`):** For clock logic and countdown calculations.

## Assets

* **Background Images:** Sourced from [Unsplash]("https://unsplash.com/").
* **Icons:** Taken from [Bootstrap Icons]("https://getbootstrap.com/").


## How to Run Locally

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/NishakaraNirmal/Digital-Clock-Count-Down-To-New-Year-Clock.git]
    ```
2.  **Open the Project:** Navigate to the cloned folder.
3.  **Launch the Clock:** Double-click the `index.html` file to open it in your web browser.
